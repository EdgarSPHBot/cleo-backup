**WHOOP API key learnings:**
- REST API base: `https://api.prod.whoop.com/developer/v1/` (confusingly named "v1" but it's current)
- Webhook model: v2 (UUID IDs) — completely separate concept from API versioning
- Sleep by UUID: `GET /developer/v1/activity/sleep/{uuid}`
- Workout by UUID: `GET /developer/v1/activity/workout/{uuid}`
- Recovery: no by-UUID endpoint — fetch collection and match on `sleep_id`
- Token expiry: 3600s — refresh logic critical
- v1 webhooks: removed/dead
